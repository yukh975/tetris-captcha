# Tetris CAPTCHA

Интерактивная CAPTCHA в стиле тетриса. Пользователь перетаскивает мышью подходящую фигуру в отверстие на игровом поле. Без ввода с клавиатуры — только мышь или тач.

## Как работает

1. Пользователь заполняет форму и нажимает «Отправить»
2. Появляется модальное окно с CAPTCHA
3. На поле 10×10 есть отверстие в форме случайной фигуры
4. Справа — несколько фигур на выбор (одна правильная, остальные — обманки)
5. Нужно перетащить правильную фигуру на поле
6. При успехе — форма отправляется, при ошибке — генерируется полностью новое задание

## Защита от ботов

- **Процедурная генерация** — каждая фигура уникальна (6-9 клеток, случайный полиомино), невозможно составить словарь всех вариантов
- **Анализ поведения мыши** — проверяется траектория, смена направлений, вариативность скорости; прямолинейное или слишком быстрое движение отклоняется
- **Минимальное время решения** — ответ быстрее 800мс считается подозрительным
- **Honeypot** — скрытое поле формы, которое боты заполняют автоматически
- **Нарастающая задержка** — при повторных ошибках пауза увеличивается
- **Лимит попыток** — после 5 неудач капча блокируется на 30 секунд
- **Одноразовые задания** — при любой ошибке генерируется новая фигура и новое поле
- **JS вынесен в отдельный файл** — логика не видна в page source HTML
