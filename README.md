# Tetris CAPTCHA

Интерактивная CAPTCHA в стиле тетриса. Пользователь поворачивает и перетаскивает мышью фигуры в подходящие отверстия на игровом поле. Без ввода с клавиатуры — только мышь или тач.

## Как работает

1. Пользователь заполняет форму и нажимает «Отправить»
2. Появляется модальное окно с CAPTCHA
3. На поле 15×15 есть **3-4 отверстия**, каждое в форме уникальной фигуры
4. Справа — 7 фигур на выбор (3-4 правильных + обманки), каждая с кнопкой поворота ↻
5. Правильные фигуры показаны **в неправильной ориентации** — нужно повернуть перед размещением
6. Нужно найти все правильные фигуры, повернуть и перетащить на поле
7. При успехе — форма отправляется, при ошибке — генерируется полностью новое задание

## Защита от ботов

- **Процедурная генерация** — каждая фигура уникальна (7-10 клеток, случайный полиомино), невозможно составить словарь
- **Поворот фигур** — правильные фигуры показаны в неправильной ориентации, бот должен визуально сопоставить и повернуть
- **Множественные цели** — нужно разместить 3-4 фигуры, а не одну
- **Анализ поведения мыши** — траектория, смена направлений, вариативность скорости
- **Минимальное время решения** — ответ быстрее 1.5с считается подозрительным
- **Honeypot** — скрытое поле формы, которое боты заполняют автоматически
- **Нарастающая задержка** — при повторных ошибках пауза увеличивается
- **Лимит попыток** — после 5 неудач капча блокируется на 30 секунд
- **Одноразовые задания** — при ошибке всё перегенерируется заново
- **JS вынесен в отдельный файл** — логика не видна в page source HTML
