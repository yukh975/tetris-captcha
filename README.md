# Tetris CAPTCHA

Интерактивная CAPTCHA в стиле тетриса. Пользователь поворачивает и перетаскивает фигуры в подходящие отверстия на игровом поле. Без ввода с клавиатуры — только мышь или тач.

## Как работает

1. Пользователь нажимает «Отправить» на форме
2. Появляется модальное окно с CAPTCHA
3. На поле 15×15 расположены **3-4 отверстия** уникальной формы
4. Вокруг поля (сверху и снизу) — 7-8 фигур, часть из которых обманки
5. Правильные фигуры показаны **в неправильной ориентации** — нужно повернуть кнопкой ↻
6. Нужно найти все правильные фигуры, повернуть и перетащить на поле
7. При любой ошибке — полный сброс с новым заданием

## Защита от ботов

- **Процедурная генерация** — каждая фигура (7-10 клеток) генерируется случайно, невозможно составить словарь
- **Поворот фигур** — правильные фигуры показаны в неправильной ориентации
- **Множественные цели** — нужно разместить 3-4 фигуры, а не одну
- **Анализ поведения мыши** — траектория, смена направлений, вариативность скорости
- **Минимальное время решения** — слишком быстрый ответ отклоняется
- **Honeypot** — скрытое поле формы, которое боты заполняют автоматически
- **Нарастающая задержка** — при повторных ошибках пауза увеличивается
- **Лимит попыток** — после 5 неудач капча блокируется на 30 секунд
- **Полный сброс при ошибке** — всё перегенерируется заново
- **JS в отдельном файле** — логика не видна в page source HTML
