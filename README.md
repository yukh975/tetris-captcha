# Tetris CAPTCHA

Интерактивная CAPTCHA в стиле тетриса. Пользователь перетаскивает мышью подходящую фигуру в отверстие на игровом поле. Без ввода с клавиатуры — только мышь или тач.

## Как работает

1. Пользователь заполняет форму и нажимает «Отправить»
2. Появляется модальное окно с CAPTCHA
3. На поле 10×10 есть отверстие в форме одной из фигур
4. Справа — несколько фигур на выбор (одна правильная, остальные — обманки)
5. Нужно перетащить правильную фигуру на поле
6. При успехе — форма отправляется, при ошибке — генерируется новое задание

## Фигуры

- **Тетромино** (4 клетки) — 7 классических фигур тетриса: I, O, T, S, Z, L, J
- **Пентамино** (5 клеток) — 12 фигур: F, I, L, N, P, T, U, V, W, X, Y, Z
- **Гексамино** (6 клеток) — 8 сложных фигур

Каждая фигура автоматически генерируется во всех уникальных поворотах.

## Особенности

- Standalone HTML — один файл, без зависимостей и сборки
- Canvas-рендеринг поля и фигур
- Drag & Drop с поддержкой мыши и touch-событий
- Обманки подбираются из фигур того же размера (сложнее отличить)
- Тёмная тема

## Запуск

Откройте `index.html` в браузере.
